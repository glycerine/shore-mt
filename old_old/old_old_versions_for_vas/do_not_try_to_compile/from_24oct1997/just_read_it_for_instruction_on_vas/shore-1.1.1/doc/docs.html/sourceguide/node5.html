Configuration management (config, tools)

These two directories are at the root of a source releases.

tools
A variety of executable shell and Perl scripts uses for configuring and building Shore from sources. Particularly noteworthy is makemake, a Perl script for creating Makefiles from Imakefiles.
config
files used by tools/makemake to generate Makefiles. Particularly noteworthy is config.h, which is used both to control the actions of makemake and as a header file included in virtually all C and C++ source files.
