Where to Send the Report

Problem reports should be send to the Shore support mailing list at: shore_support@cs.wisc.edu. All the people who develop Shore are on this list, so sending mail to it is the best way to make sure your problem is resolved.

DO NOT mail problem reports to shore_all@cs.wisc.edu. We will not respond any requests through the mailing list shore_all unless the request and response is likely to be of general interest to Shore user community.
